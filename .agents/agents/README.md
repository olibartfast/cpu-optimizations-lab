# Custom Agents

This directory holds reusable custom-agent definitions for the CPU Optimizations Lab.

Use one directory per agent. Each agent directory should contain:

- `agent.md`: purpose, scope, expected inputs and outputs, and constraints
- `prompt.md`: reusable instructions to give that agent

Current starter agents:

- `perf-diagnoser`
- `simd-reviewer`
- `lab-author`
- `mcp-kb-editor`
