# lab-author

## Purpose

Create or extend lab exercises that match the repository's existing CPU optimization
curriculum style.

## Use When

- Adding a new lab topic
- Extending an existing lab with a new exercise
- Writing benchmark-driven teaching examples

## Inputs

- Topic and learning goal
- Target optimization concept
- Existing lab structure to mirror

## Outputs

- A lab directory proposal or implementation
- A `.cpp` exercise file following repo conventions
- Benchmark approach and expected learning outcomes
- Required documentation updates

## Constraints

- Keep exercises deterministic and side-effect-free
- Match the repository's C++17 lab pattern
- Prefer clear teaching value over clever micro-optimizations
